Next Steps

Die nächsten Schritte um den Test umzusetzen wäre, dass der Test genau protokolliert würde. Es müssten Kriterien beschlossen werden an Hand derer der Test ausgewertet werden kann. Als nächster Schritt würden dann Testpersonen eingeladen werden um den Test durchzuführen. Das Ziel des Tests muss im Vorhinein protokolliert werden.

Der Test müsste protokolliert und dokumentiert werden, beispielsweise per Video, um den Test anschließend auswerten zu können. Der Test müsste nun hunderte Male durchgeführt werden um ein aussagekräftiges Ergebnis zu erhalten. Auch die Testpersonen müssten bei den verschiedenen Tests ausgetauscht werden. Erst durch eine große Anzahl von Tests könnte das Ergebnis etwas aussagen über die aufgestellte Arbeitsthese. Bei einer einmaligen Durchführung könnte jedes Ergebnis bloßer Zufall sein. Somit ist eine große Anzahl an Wiederholung des Tests notwendig.

Nach der Durchführung der Tests müssten diese anschließend ausgewertet werden. Das Ergebnis der Tests würde an dem Verständnis der Person C Bei einer Mehrheit der Tests in dem Person C den Strauß näher an dem Musterblumenstrauß nachbauen könnte, nachdem die Beschreibung mit Sprache oder mit Sprache und Bild erfolgt, würde das ein Ergebnis des Tests liefern.
